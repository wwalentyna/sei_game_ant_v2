# SAI Governance Challenge — bilingual

React + TypeScript + Vite project for GitHub Pages.

This version contains both language versions:

- German
- English

The start screen lets players choose their language.

## Deploy on GitHub Pages

1. Upload all files in this folder to your GitHub repository.
2. Go to `Settings -> Pages`.
3. Set `Source` to `GitHub Actions`.
4. Go to `Settings -> Actions -> General`.
5. Set workflow permissions to `Read and write permissions`.
6. Push to `main` or run the workflow manually from the `Actions` tab.

Open your GitHub Pages link after the workflow is green.

If the page was blank before, open it with `?v=3` at the end once, for example:

```text
https://YOUR-USERNAME.github.io/YOUR-REPOSITORY-NAME/?v=3
```
